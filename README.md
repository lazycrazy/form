## Build Setup


```bash
# clone the project
git clone git@github.com:self-heal/form.git

# enter the project directory
cd form

# install dependency
npm install

# develop
npm run dev
```

This will automatically open http://localhost:9528

## Build

```bash
# build for test environment
npm run build:stage

# build for production environment
npm run build:prod
```
